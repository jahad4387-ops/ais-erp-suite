# AIS ERP Suite PRD

## 1. 产品目标

AIS ERP Suite 是一个完整的企业财务与业务核算系统。它以教材目录覆盖的会计信息系统为功能边界，支持从业务单据、凭证、账簿到报表的全链路闭环，并允许 AI 和外部 Agent 在受控权限、审批与审计下协助工作人员完成记账、对账、分析和结账工作。

产品必须满足四个原则：

1. 业务可追溯：每一笔账可以追溯到单据、附件、审批、凭证、账簿和报表取数。
2. 会计可验证：凭证借贷平衡，期间、科目、辅助核算、结账状态和反向流程可控。
3. Agent 可控：AI/Agent 默认只能生成草稿、建议和检查结果；正式记账、付款、结账等高风险动作必须审批。
4. 实施可验收：每个教材章节对应明确功能、数据、测试用例和上线验收项。
5. 部署可切换：同一套系统可部署在局域网、私有云/VPN 或公网 SaaS，不因部署方式改变核心业务规则。

## 2. 用户与角色

| 角色 | 典型工作 |
| --- | --- |
| 系统管理员 | 建账套、设权限、初始化参数、备份恢复、监控审计 |
| 财务主管 | 审核凭证、审批付款、检查结账、查看报表 |
| 总账会计 | 填制凭证、记账、查询账簿、期末处理 |
| 出纳 | 现金银行日记账、收付款、银行对账、出纳签字 |
| 采购会计 | 采购发票、应付确认、付款核销、供应商分析 |
| 销售会计 | 销售发票、应收确认、收款核销、账龄分析 |
| 仓库/存货会计 | 出入库、盘点、成本核算、收发存分析 |
| 薪资会计 | 工资项目、工资计算、工资分摊、个税/社保接口 |
| 固资会计 | 资产卡片、折旧、转移、处置、盘点 |
| 业务经理 | 采购、销售、库存、回款、成本和毛利看板 |
| Agent 操作员 | 通过 Codex、Claude Code 或内部 Agent 发起受控业务任务 |

## 3. 功能范围

### 3.1 系统管理与基础档案

- 多账套、多组织、单本位币、多会计准则配置。
- 远期规划 (Phase 7)：多币种及汇兑损益、复杂多公司合并报表暂不在 Phase 1~6 范围内。
- 会计期间、启用期间、结账状态、反结账权限。
- 部门、人员、客户、供应商、仓库、存货、资产、项目、现金流项目、结算方式。
- 操作员、角色权限、字段级权限、数据范围权限、Agent 工具权限、审批流。
- 编码规则、单据编号规则、单据类型、单据模板、凭证字、摘要模板、常用凭证。
- 单据设置：字段配置、状态流、打印模板、导出模板、审批模板。
- 操作日志、审计日志、附件归档、备份恢复。
- 系统日常高级维护：数据检测与健康检查、历史数据归档与清理、重建索引与性能优化。
- 部署模式配置：支持内网、私有云/VPN、公网 SaaS 的访问入口、存储和安全策略配置。

### 3.2 总账

- 会计科目：科目编码、方向、余额方向、辅助核算、现金流属性、受控系统。
- 期初余额：本位币、外币、数量、辅助核算期初，试算平衡。
- 凭证：新增、复制、草稿、提交、审核、出纳签字、记账、冲销、作废、反审核。
- 凭证分录：借方、贷方、摘要、科目、辅助核算、数量、外币、现金流项目。
- 账簿：总账、明细账、日记账、科目余额表、试算平衡表、多栏账、辅助账。
- 银行对账：银行日记账、银行对账单、自动勾对、手工勾对、余额调节表。
- 期末：自动转账、期间损益结转、汇兑损益、凭证编号整理、月末结账。
- 管控：已记账凭证不可物理删除；反向操作必须保留审计记录。

### 3.3 采购与付款

- 供应商档案、采购部门、采购员、付款条件、税率、结算方式。
- 采购申请、采购订单、到货单、采购入库单、采购发票、采购退货。
- 应付账款、预付款、付款申请、付款单、付款审批、付款核销。
- 暂估入库、发票到票回冲、采购成本入账、进项税处理。
- 供应商往来账、账龄分析、付款计划、采购价格分析。
- 与总账、存货、现金银行联动生成凭证。

### 3.4 销售与收款

- 客户档案、信用额度、销售部门、销售员、收款条件、价格策略。
- 销售报价、销售订单、发货单、销售出库单、销售发票、销售退货。
- 应收账款、预收款、收款单、坏账准备、收款核销。
- 收入确认、销项税、销售成本结转、销售毛利分析。
- 客户往来账、账龄分析、回款计划、催收清单。
- 与总账、存货、现金银行联动生成凭证。

### 3.5 存货核算与库存

- 存货档案、分类、计量单位、换算率、批次、序列号、保质期。
- 仓库、货位、库存期初、库存上下限。
- 采购入库、销售出库、材料出库、产成品入库、其他出入库、调拨、盘点。
- 成本方法：移动平均、先进先出、个别计价、月末一次加权。
- 暂估成本、出库成本计算、成本调整、跌价准备。
- 库存台账、收发存汇总、库存明细、库龄、呆滞料、存货周转。

### 3.6 薪资

- 员工档案、薪资类别、工资项目、公式、部门分摊。
- 考勤/绩效接口、社保公积金、个税计算接口或规则配置。
- 工资计算、工资审核、工资发放、银行代发文件。
- 工资分摊凭证、工资费用分析、员工工资台账。
- 工资期末处理：工资分摊、费用归集、凭证生成、月末检查。

### 3.7 固定资产

- 资产类别、卡片、使用部门、责任人、使用状态。
- 原值、累计折旧、净值、预计净残值、折旧年限、折旧方法。
- 新增、减少、转移、原值变动、使用状态变动、报废处置。
- 折旧计提、折旧分摊凭证、资产盘点、减值准备。
- 固定资产明细账、折旧表、资产结构分析。
- 固定资产期末处理：折旧计提、折旧凭证、资产与总账对账。

### 3.8 会计报表

- 报表模板：资产负债表、利润表、现金流量表、所有者权益变动表。
- 公式引擎：科目取数、辅助核算取数、期间取数、跨账套汇总、自由格式单元格公式。
- 报表编制、审核、锁定、导出 Excel/PDF。
- UFO 报表式能力：模板复制、格式设计、单元格取数、报表输出。
- 管理报表：采购、销售、存货、应收、应付、现金流、经营分析。

### 3.9 AI/API/Agent

- OCR：发票、银行流水、合同、付款申请、收据、工资表。
- AI 记账：生成凭证草稿、科目建议、辅助核算建议、摘要建议。
- AI 检查：重复报销、异常金额、科目错配、税率异常、账龄风险。
- AI 分析：报表解读、经营分析、现金流预测、催收建议、库存补货建议。
- Agent API：提供 dry-run、幂等、审批、审计、证据链和回滚策略。
- 外部系统集成 API：提供标准化进销存与支付对接接口，支持 CRM、电子商务平台等外部业务系统的集成对接。

## 4. 非功能需求

- 安全：最小权限、双人复核、敏感字段脱敏、密钥托管。
- 审计：所有单据、凭证、审批、AI 建议、Agent 动作留痕。
- 性能：中小企业单账套 100 万分录查询可在秒级返回常见账表。
- 可用性：月末结账、凭证审核、账簿查询必须支持批量操作。
- 可迁移：支持从 Excel、旧桌面工具、其他 ERP 导入基础档案和期初。
- 可扩展：模块化单体优先，后续可拆 AI worker、报表服务、集成服务。
- 多人同步：多用户通过中心服务器和中心数据库共享数据，附件上传、OCR 结果、审批状态和账务状态应及时同步。
- 部署弹性：同一套代码支持局域网内网部署、私有云/VPN 部署和公网 SaaS 部署。
- 并发控制：同一单据或凭证被多人同时编辑时必须通过版本号、状态机或锁机制防止覆盖。

## 5. 关键验收标准

- 总账从期初、凭证、审核、记账、账簿、报表、结账完整闭环。
- 采购、销售、存货、薪资、固定资产均可独立初始化、日常处理、期末处理。
- 业务单据可以自动或手动生成凭证，并可反查来源。
- AI 生成的凭证建议必须可解释、可修改、可拒绝、可追溯。
- Agent 不能绕过权限和审批直接执行正式财务动作。
- 已结账期间不能新增正式凭证，除非走授权反结账流程。
- 同一套系统可以在内网和公网环境部署，且 Agent API、权限、审计、实时同步行为一致。
